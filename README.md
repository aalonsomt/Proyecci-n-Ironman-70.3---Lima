# Proyección-Ironman-70.3---Lima
Este es un proyecto que personalmente tiene un significado muy profundo ya que actualmente me preparo para mi primer Ironman 70.3 en 2027 y esta información será una herramienta que me servirá para analizar los eventos anteriores y aprovechar la información detrás de los números.

## Objetivo

Centralizar y analizar los resultados históricos del Ironman 70.3 Lima para facilitar la consulta, comparación y benchmarking de atletas, categorías y países.

## Alcance

- Consolidación de resultados históricos.
- Limpieza y estandarización de datos.
- Análisis exploratorio.
- Cálculo de métricas y percentiles.
- Dashboard interactivo.
